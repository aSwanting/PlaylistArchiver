How to get an API key:

  1. Go to the Google Cloud Console.
  2. Create a new project or select an existing one.
  3. Navigate to APIs & Services > Credentials.
  4. Click Create Credentials > API key.
  5. Copy the generated API key and paste it into the input field.

How to find the playlist ID:

  1. Go to YouTube and open the playlist.
  2. Check the URL. The playlist ID is the part after list=, e.g., PL123456789ABCDEFG.
